在program类的main函数中添加以下代码

``` csharp
bool CreateNew;
System.Threading.Mutex mutex = new System.Threading.Mutex(false, "程序的名称", out CreateNew);
if (!CreateNew)
{
    MessageBox.Show("程序正在运行中!", "提示");
    Application.Exit();
    return;
}
```
